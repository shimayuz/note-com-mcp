# 🚀 note MCP Server クイックスタート

**このファイル1枚だけで設定完了できます。**

---

## 📋 やることは3つだけ

### 1️⃣ リポジトリをダウンロード

```bash
git clone https://github.com/shimayuz/note-com-mcp.git
cd note-com-mcp
```

### 2️⃣ セットアップ実行

**Mac/Linux:**
```bash
npm run setup
```

**Windows (PowerShell):**
```powershell
npm run setup:win
```

### 3️⃣ Cursor を再起動

設定完了！「noteで記事を検索して」と試してみてください。

---

## ⚠️ Node.js が入っていない場合

セットアップ前に Node.js をインストール：

- **Mac**: `brew install node`
- **Windows**: https://nodejs.org/ からダウンロード

---

## 💡 認証について

初回起動時にブラウザが開くので、note.com にログインしてください。  
自動でセッション情報が保存されます。
